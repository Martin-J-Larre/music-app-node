Api Music App Node

User CRUD

Register

Request Method: POST
https://music-app-node.onrender.com/api/user/register

Login

Request Method: POST
https://music-app-node.onrender.com/api/user/login

Get user profile

Request Method: GET (add token in headers Authorization)
https://music-app-node.onrender.com/api/user/profile/${user id}

Update user Auth

Request Method: PUT (add token in headers Authorization)
https://music-app-node.onrender.com/api/user/update

Upload user avatar

Request Method: POST (add token in headers Authorization)
https://music-app-node.onrender.com/api/user/upload

Get user avatar

Request Method: GET
https://music-app-node.onrender.com/api/user/avatar/${ filename }
